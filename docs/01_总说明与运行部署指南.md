## 6. 风险与常见问题

### 问题 1：Colab 明明改了文件，还是报旧错
原因通常是：
- 运行时缓存了旧模块；
- `sys.path` 指向旧目录；
- 同时存在 `Assignment1_2026` 和 `Assignment1_2026_fixed`。

处理：
- 重启运行时；
- 清理 `sys.path`；
- 只使用 fixed repo。

### 问题 2：checkpoint 保存报 lambda scheduler 错误
原因是匿名 lambda 不可 pickle。

处理：
- 当前 notebook 默认使用 `step` scheduler；
- 不要在提交版 notebook 中改回匿名 lambda 方案。

### 问题 3：保存失败 / Drive 不同步
处理：
- 确认 notebook 存在 Drive 目录下；
- 重要结果立即复制到本地或 git commit；
- 训练结束后检查 `_model*`、`_log*` 是否确实落盘。

### 问题 4：本地能跑，Colab 不能跑
说明还有隐藏本地依赖。必须回到干净 Colab 环境复现并修正。
